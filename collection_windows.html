Uso

Esta es una colección de interfaces de ventana estilo pixel art que puedes usar libremente en tu proyecto. Si deseas contribuir con nuevas variaciones para ampliar esta colección (en cuyo caso se te dará acceso a la versión editable en Aseprite), debes demostrar que son de tu propiedad. Se te otorgarán los créditos correspondientes.

Algunas de estas ventanas fueron encontradas en internet. Si alguna es de tu autoría, contáctame con pruebas para que te dé el crédito adecuado o la elimine.

Si tienes sugerencias para mejorar o agregar nuevas variantes, siéntete libre de compartirlas.

Características

  • Diferentes estilos de ventanas para interfaces retro.
  • Formato editable en Aseprite para personalización.
  • Actualizaciones constantes con mejoras y nuevas variantes.
